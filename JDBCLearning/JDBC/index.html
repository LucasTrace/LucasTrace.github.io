Definition of JDBC 定义

Java Database Connectivity

JDBC是SUN公司制定的一套接口（interface）

接口都有调用者和实现者，面向接口调用、面向接口写实现对象类，这都属于面向接口编程

使用statement来执行sql语句

executeUpdate(sql)函数专门执行DML语句，返回一个int，表示改变的数据条数

executeQuery(sql)函数专门执行DQL语句，返回一个ResultSet，便于后续使用next()函数进行遍历输出

DML和DQL的以及其他SQL语句类型在之前的blog中有过介绍 Classification of SQL Statements

使用PreparedStatement

对sql语句框架进行预编译，可以防止sql注入

需要传入的用户信息提前使用占位符? 之后再使用setInt(),setString()…等函数传入用户信息，但应注意JDBC中的下标句是从1开始的，于常规的数组下标从0开始有所不同。

Statement&PreparedStatement

Statement特点：编译一次执行一次
PreparedStatement特点：编译一次执行多次 ，且会在编译阶段作类型的安全检查
