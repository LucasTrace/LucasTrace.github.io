python practice - 1 StudentInfoSystem

将python文件打包成一个exe文件

1.将python安装文件下的C:\……Python\Python38\Scripts路径添加到系统环境变量中

2.在Windows dos命令窗口中，输入pip install PyInstaller，等待第三方模块PyInstaller安装完成

3.在Windows dos命令窗口中， pyinstaller -F thepathofXXX.py文件（-F表示只生产一个exe文件）
