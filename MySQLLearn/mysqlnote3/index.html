MySQL note-Three

连接查询

笛卡尔乘积现象

内连接

A与B表进行内连接，AB表地位平等，查询出能匹配的数据

等值连接

加条件过滤，避免笛卡尔乘积现象

非等值连接

自连接

外连接

A与B表进行外连接，AB中有一个表是主表，一个表是副表，主要查询主表中的数据，捎带查询副表，当副表没有和主表上的数据匹配时，附表自动模拟出null与之匹配。即：将主表数据无条件全部查询。

三表连接：A join B join C A先于B连接，其结果再与C连接

全连接

不常用

子查询

UNION

MySQL中常见数据类型

int 整数型 bigint 长整型 float 浮点型 char 定长字符串

varchar 不定长字符串(max255字符) （对应Java中的StringBuilder/StringBuffer）

date 日期类型 （对应Java中的 java.sql.date）

BLOB 二进制大对象（存储图片、视频等流媒体信息，对应Java中的Object） Binary Large Object

CLOB 字符大对象（存储大文本，如4G字符串，对应Java中的Object） Character Large Object

(插入BOLB或者CLOB，不能使用简单的insert 语句，需要使用Java里的IOStream)
