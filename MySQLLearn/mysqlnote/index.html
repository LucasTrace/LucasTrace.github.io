MySQL note-ONE

1.Definitions of sql,DB and DBMS & Relationship among them

DB：

​ DataBase（数据库、数据库实际上在硬盘上以文件的形式存在）

DBMS：

​ DataBase Management System（数据库和管理系统，常见的有MySQL Oracle DB2 Sybase SqlServer……

SQL：

​ 结构化查询语言，是一门标准通用的语言。标准的sql适用于所有数据库产品。

​ SQL属于高级语言。只要能看懂英语单词，写出来的sql语句，就可以看懂其意思。

​ SQL语句在执行的时候，内部也会先编译（由DBMS完成，我们看不见），然后执行sql。

Summary：

​ DBMS负责执行sql语句，通过执行sql语句在操作DB当中的数据。

​ DDMS - (执行) - > SQL - (操作) - > DB

2.What is table?

table：

​ 是数据库的基本组成单元，所有的数据都已表格的形式组织，目的是可读性强。

​ one table contains rows and columns

​ row： 数据/记录（data）

​ column：字段（column）

table example：

Number(int) Name(varchar) Age(int)
001 Tom 19
002 Jack 20

3.Classification of SQL Statements

DQL（Data Query Language 数据查询语言）：

​ 查询语句，凡是select语句都是DQL。

DML（Data Manipulation Language 数据操作语言）:

​ insert delete update，对表中的数据进行增删改。

DDL（Data Definition Language 数据定义语言）:

​ create drop alter, 对表结构的增删改。

TCL（Transaction Control Language 事务控制语言）:

​ commit提交事务，rollback回滚事务。

DCL（Data Control Language 数据控制语言）:

​ grant授权、revoke撤销权限等。

4.XXX.sql

​ 这样的文件被称作“sql脚本”

​ sql文件过大时将无法打开，可以使用source命令，执行sql脚本，完成初始化
